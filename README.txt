Tez Computers package (online-ready, Blue & White theme)
Email: tezitpvtltd@gmail.com

Contents:
- tez-backend/    (Node.js + SQLite backend)
- tez-frontend/   (React/Vite frontend skeleton)
- invoice-template.html
- google-sheets-csvs/

Quick start (backend):
1) cd tez-backend
2) npm install
3) create .env (JWT_SECRET=your_secret)
4) npm run migrate
5) npm start

Quick start (frontend):
1) cd tez-frontend
2) npm install
3) npm run dev

Default admin: created after you set up users manually (run migrate or create user)
